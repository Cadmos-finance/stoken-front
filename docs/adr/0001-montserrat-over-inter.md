# Use Montserrat as the public-site typeface, reverting the prior Inter decision

The earlier homepage refinement PRD (`docs/prd/stoken-homepage-visual-refinement.md`, "Implementation Decisions") deliberately replaced Montserrat with Inter for a cleaner SaaS/institutional feel. The 2026-05-19 graphical-chart update from the client reverses that call: all chart artefacts in `update/` are designed in Montserrat, the brand's `Livraison/Fonts/Montserrat` family is the canonical type, and the client's brief was explicitly "match the graphical chart as accurately as possible." We are therefore switching the site type system back to Montserrat (variable weights from the supplied family) so the rendered pages match the chart exactly. Geist Mono stays as the monospace pair for data/labels.
